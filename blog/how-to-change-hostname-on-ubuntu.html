How to Change Hostname on Ubuntu

Table of Contents

  • 1. 临时更改
  • 2. 永久更改
  • 3. 手动修改（不推荐，需重启）

1. 临时更改

hostname newHostName

2. 永久更改

hostnamectl set-hostname newHostName

更改后新 session 连接即可查看到变化

3. 手动修改（不推荐，需重启）

需要修改以下两个文件 

sudo emacs /etc/hostname
sudo vim /etc/hosts

接着重启才能生效 

sudo systemctl reboot
